To get started with this application
you will need your own Yandex API_KEY.

Visit here to get your API key.
https://passport.yandex.com/auth?origin=translate&retpath=https%3A%2F%2Ftranslate.yandex.com%2Fdevelopers%2Fkeys

Steps:
1. Clone this repo.
2. Create a config folder in root.
3. Create config.js file and incude the following..

API_KEY = "Your API Key that you received from Yandex".


4. Run it locally, translate to and from over five different languages!
